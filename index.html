MiniCodeEditor

An online HTML/CSS/JS playground in 64 & 149 bytes

Inspired by Codepen, JSFiddle and dabblet
golfed by xem, p01, subzey, aemkei, rlauck, bburky, mmastrac and justecorruptio




Full (149b): Edit HTML, CSS and JS, the result is updated in real time 
<iframe onload='id++<3?outerHTML+=`<textarea oninput="f.srcdoc=t3.value+\`<script>\${t1.value}</script><style>\`+t2.value"id=t${id} rows=9>`:id="f"'>



Mini (64b): HTML only 
<textarea oninput=f.srcdoc=value rows=9></textarea><iframe id=f>
